1. git fetch <원격지명>
- 원격지 주소가 아닌 이름(origin 같은)을 입력
- 원격지의 모든 브랜치와 커밋 정보를 가져온다
- 즉, 원격지의 최신 정보를 로컬에 적용
- 로컬에 바로 적용되지 않기 때문에 적용하려면 git reset로 적용한다

2. git pull<원격지명/브랜치> <로컬 브랜치명)
- 원격지의 브랜치ㅗ아 커밋 정보를 가져와서 로컬의 브랜치에 바로 병합
- 즉, 원격지의 정보를 바로 로컬에 적용시 사용
- 병합시 충돌이 발생할 수 있어 별로 권장하지 않음
- 모든 브랜치는 가져올 수 없고 지정한 브랜치가 가져온다
